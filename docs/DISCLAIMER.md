# Disclaimer

## NekoTerm

El objetivo principal de este proyecto es correlacionar automatización y eficiencia para extraer lo que se proporciona a un usuario en internet. Todo el contenido disponible a través del proyecto es alojado por fuentes externas no afiliadas.

**Cualquier contenido servido a través de este proyecto es accesible públicamente. Si un sitio está listado en este proyecto, el código ya es público. Tome las medidas necesarias para contrarrestar los métodos utilizados para extraer contenido de su sitio.**

Piense en este proyecto como su navegador web habitual, pero más directo y específico. Mientras que un navegador promedio hace cientos de solicitudes para obtener todo de un sitio, este proyecto solo realiza las solicitudes necesarias para obtener el contenido servido por los sitios.

**Este proyecto es para ser utilizado bajo el propio riesgo del usuario, en función de su gobierno y leyes locales.**

Este proyecto no tiene control sobre el contenido que sirve. El uso de contenido con derechos de autor por parte de los proveedores no será responsabilidad del desarrollador. Es el propio riesgo del usuario.

## DMCA y Reclamaciones de Derechos de Autor

**Un navegador es una herramienta, y la malicia de la herramienta se basa directamente en el usuario.**

Este proyecto utiliza mecanismos de acceso al contenido del lado del cliente. Por lo tanto, las reclamaciones por infracción de derechos de autor o DMCA deben dirigirse a los sitios asociados por el notificador correspondiente.

**No hostigue al mantenedor.** Las reclamaciones legítimas deben dirigirse a los sitios que alojan el contenido, no a este proyecto.

## Responsabilidad del Usuario

El usuario es el único responsable de:

- Verificar que el uso de esta herramienta cumple con las leyes locales de propiedad intelectual
